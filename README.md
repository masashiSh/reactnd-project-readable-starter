# Readable Project

## Instructions

 1. clone the Project - `git clone https://github.com/masashiSh/reactnd-project-readable-starter.git`
 2. install the dependencies and start 
    * API server
        - `cd api-server`
        - `npm install`
        - `node server`
    * front-end
        - `cd frontend`
        - `npm install`
        - `npm start`
